# Store App

## Features

    - User Login, Registration, JWT TOKEN Auth
    - CURD Store
    - CURD Store Items

### Project setup

```
mkdir store
cd store
python -m venv venv
source venv/bin/activate
.\venv\Scripts\activate.bat (Windows)
git clone git_url
cd store-api-flask
run flask db init
run flask db migrate
run flask db upgrade
python app.py
```
